# 🚀 Quick Start Guide - ResumeGenie Add-on

**Get text selection working in 5 minutes!**

## ⚡ **Super Quick Setup:**

### **1. Go to Google Apps Script**
- Visit: [script.google.com](https://script.google.com)
- Sign in with your Google account

### **2. Create New Project**
- Click **"New Project"**
- Delete the default code
- Copy-paste the `Code.gs` content
- Save (Ctrl+S)

### **3. Deploy as Add-on**
- Click **"Deploy"** → **"New deployment"**
- Choose **"Add-on"**
- Click **"Deploy"**

### **4. Install in Google Docs**
- Open any Google Doc
- **Extensions** → **Add-ons** → **Manage add-ons**
- Find **ResumeGenie** and click **"Install"**

### **5. Test It!**
- Select some text in your Google Doc
- **Extensions** → **ResumeGenie** → **Extract Selected Text**
- **BOOM!** Your selected text is extracted! 🎯

## 🔍 **What You'll See:**

After installation, you'll have a new menu:
```
Extensions → ResumeGenie
├── Extract Selected Text    ← This is what you want!
├── Extract All Text
└── Show Selection Info
```

## ✅ **Success Indicators:**

- ✅ Add-on appears in Extensions menu
- ✅ No error messages during installation
- ✅ Can select text and extract it
- ✅ Console shows "Document initialized successfully"

## 🆘 **If Something Goes Wrong:**

1. **Check the console** in Google Apps Script
2. **Make sure you're in a Google Doc** (not Google Drive)
3. **Grant all permissions** when prompted
4. **Try refreshing** the Google Doc
